# Commercial Building Permit Package Checklist

## Overview

This comprehensive checklist covers all components of a complete commercial building permit application package. The checklist is organized by discipline and includes jurisdiction-specific guidance for forms, drawings, calculations, and supporting documentation.

**Integration with QA Workflow:**
Permit requirements inform which QA rules are most critical for compliance. This checklist ensures the permit package is complete before submission, reducing plan review cycles and expediting project approval.

**How to Use This Checklist:**
- [ ] Mark each item as Complete, In Progress, or N/A
- Gather all required items before submitting to jurisdiction
- Verify all items are current (dates within 6 months of submission)
- Obtain all required professional seals and signatures before submission
